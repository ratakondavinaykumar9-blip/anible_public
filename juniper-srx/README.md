## Requirements

- Ansible 2.9+
- Juniper Collection: `ansible-galaxy collection install junipernetworks.junos`
- Python: `junos-eznc`

## Usage

```bash
ansible-playbook -i inventory security/zones.yml
ansible-playbook -i inventory security/policies.yml --check
ansible-playbook -i inventory site.yml --limit srx-01
```
