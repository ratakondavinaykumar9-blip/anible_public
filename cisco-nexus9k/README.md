## Requirements

- Ansible 2.9+
- Cisco NX-OS Collection: `ansible-galaxy collection install cisco.nxos`
- Python libraries: `pyntc`, `pynxos`

## Usage

```bash
# Run all basic configuration
ansible-playbook -i inventory basic/site.yml

# Run specific playbook
ansible-playbook -i inventory interfaces/vlans.yml

# Run with tags
ansible-playbook -i inventory site.yml --tags "interfaces,routing"

# Check mode (dry run)
ansible-playbook -i inventory site.yml --check
```
