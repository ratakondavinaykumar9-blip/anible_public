# Cisco IOS/IOS-XE Ansible Playbooks

Production-ready Ansible automation for Cisco IOS and IOS-XE switches and routers.

## Structure

```
cisco-ios/
├── inventory
├── group_vars/
├── basic/                 # Hostname, DNS, NTP, SNMP, Users, SSH, Domain
├── interfaces/            # VLANs, Trunks, Access, Port-channels, Loopbacks
├── routing/               # OSPF, OSPFv3, BGP, EIGRP, Static, Redistribution
├── switching/             # VLANs, VTP, STP, LACP, HSRP/VRRP/GLBP
├── acl/                   # Standard, Extended, Time-based, Object Groups
├── nat/                   # Static, Dynamic, PAT, NAT64
├── vpn/                   # DMVPN, IPsec, GRE tunnels
├── monitoring/            # NetFlow, SPAN, RSPAN, Logging, SNMP traps
├── ha/                    # HSRP, VRRP, GLBP, StackWise
├── backup/                # Config backup/restore, Archival
├── security/              # AAA, 802.1X, Port-security, DHCP snooping
└── qos/                   # QoS policies, Class maps, Policy maps
```

## Requirements

- Ansible 2.9+
- Cisco IOS Collection: `ansible-galaxy collection install cisco.ios`
- Python: `pyntc`

## Supported Platforms

- Cisco Catalyst Switches (2960, 3850, 9200, 9300, 9400, 9500)
- Cisco ISR Routers (4000, 4300, 4400 series)
- Cisco ASR Routers (1000, 9000 series)

## Usage

```bash
ansible-playbook -i inventory basic/site.yml
ansible-playbook -i inventory interfaces/vlans.yml --check
ansible-playbook -i inventory routing/ospf.yml --tags "area0"
```
