# Dell EMC PowerSwitch Ansible Playbooks

Production-ready Ansible automation for Dell EMC PowerSwitch series (OS6, OS9, OS10).

## Structure

```
dell-emc/
├── inventory
├── group_vars/
├── basic/                 # Hostname, DNS, NTP, SNMP, Users, SSH
├── interfaces/            # VLANs, Trunks, Port-channels, LAG
├── routing/               # OSPF, BGP, Static routes, VRF
├── switching/             # VLANs, STP, LACP, MLAG
├── acl/                   # IP ACLs, MAC ACLs, Time-based ACLs
├── monitoring/            # NetFlow, sFlow, Logging, SPAN
├── poe/                   # PoE for VoIP devices
├── stacking/              # Switch stacking
├── backup/                # Config backup/restore
└── security/              # AAA, Port-security, DHCP snooping
```

## Requirements

- Ansible 2.9+
- Dell OS Collections:
  - `dellemc.os6` (OS6 switches)
  - `dellemc.os9` (OS9 switches)
  - `dellemc.os10` (OS10 switches)

## Supported Platforms

- PowerSwitch N-Series (OS6)
- PowerSwitch S-Series (OS9)
- PowerSwitch Z-Series (OS10)

## Usage

```bash
ansible-playbook -i inventory basic/hostname.yml
ansible-playbook -i inventory interfaces/port-channel.yml --check
```
